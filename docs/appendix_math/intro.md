# 附录 E：数学基础

本附录的目标读者是已经具备基本微积分和线性代数背景、但需要对书中涉及的数学工具做系统性回顾的学习者。我们不追求面面俱到，而是围绕强化学习中最核心的四个数学领域展开，每个概念均从几何直观或物理动机出发，给出形式化定义，最后说明其在本书正文中的具体出处。

## 四个数学领域

本附录由以下四节组成，覆盖强化学习中出现频率最高的数学概念：

| 节                                          | 主题                                             | 对应正文章节 |
| ------------------------------------------- | ------------------------------------------------ | ------------ |
| [E.1 线性代数](./linear-algebra)            | 向量、矩阵、点积、特征值、范数                   | 第 3–4、9 章 |
| [E.2 概率与统计](./probability-statistics)  | 条件概率、贝叶斯、期望、方差、常用分布、蒙特卡洛 | 第 3、5–6 章 |
| [E.3 微积分与优化](./calculus-optimization) | 导数、链式法则、梯度下降、Adam、凸优化           | 第 5–6、8 章 |
| [E.4 信息论](./information-theory)          | 熵、KL 散度、交叉熵                              | 第 6–7、9 章 |

## 使用建议

本附录适合按需查阅，而非一次性通读。推荐的使用方式如下：

1. **随用随查。** 阅读正文时遇到不熟悉的数学符号或推导步骤，可回到本附录对应章节。
2. **分层阅读。** 每个概念按"几何直观 → 形式化定义 → 在 RL 中的具体含义"三个层次展开。读者可根据自身需求选择停留的深度。
3. **公式速查。** 每节末尾附有该领域核心公式的汇总表，供快速检索。

如果读者此前未曾系统学习概率论，建议先阅读 [E.2 概率与统计](./probability-statistics)，因为它是理解本书大部分章节的基础。
